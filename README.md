# 🧬 Bomba Na⁺/K⁺ ATPasa - Simulación Interactiva

Simulación web interactiva del mecanismo Post-Albers de la bomba sodio-potasio (Na⁺/K⁺-ATPasa), basada en investigación científica actualizada y estructuras cryo-EM.

## 🚀 [Ver Demo en Vivo](https://adolfoisla.github.io/bomba-sodio-potasio/)

## ✨ Características

- **6 pasos del ciclo Post-Albers** con transiciones animadas
- Visualización de iones Na⁺ y K⁺ moviéndose según gradientes de concentración
- Compuertas citoplasmática y extracelular que se abren/cierran en tiempo real
- Sitio de fosforilación animado (residuo Asp del dominio P)
- Modo automático con velocidad ajustable
- Tooltips informativos al pasar el cursor
- Estadísticas de ciclos, iones transportados y ATP consumido
- Diagrama del ciclo con indicador de paso actual

## 🎓 Basado en investigación científica

- Mecanismo Post-Albers (Albers 1967, Post 1969)
- Estructuras cryo-EM de la ATPasa α3 humana (Nature 2022)
- StatPearls: Physiology, Sodium Potassium Pump
- eLife: Selectivity controlled by binding site protonation

## 🛠️ Tecnologías

- HTML5 Canvas
- Vanilla JavaScript (sin dependencias)
- CSS3 con animaciones
- Responsive design

## 💡 Uso

1. Haz clic en **Siguiente** o en la proteína para avanzar paso a paso
2. Activa **Auto** para ver el ciclo completo automáticamente
3. Pasa el cursor sobre iones, proteína y membrana para más información
4. Ajusta la velocidad con el slider
